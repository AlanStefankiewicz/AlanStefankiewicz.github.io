3 ways to learn programming

BUY IT ON AMAZON

Books - no matter what you are willing to learn, there always will be some books to help you.
I don't think it's the most effective way to learn programming, but I like to let my eyes rest from the monitor from time to time.
You can also find a lot of free books online, I just found this list on GitHub: free-programming-books

Online tutorials and courses - the best learning method in my opinion.
Rewriting code is the best way to learn and practice to code. Rewriting the code from the website is easier than rewriting it from the book, but you have to rewrite the code instead of copying it.
Just type in for example: "HTML5 tutorial" in google and you will find a lot of free tutorials and courses like:
- www.w3schools.com
- www.codecademy.com
- www.udemy.com
- massive list of tutorials found on GitHub

Youtube - Watching videos is not the best way to learn. It's harder to focus watching video than reading and your brain works better when you read.
You can find plenty of video tutorials on youtube, here are some of the more popular coding channels:
- freeCodeCamp
- Derek Banas
- tutor4u
- LearnCode.academy
- thenewboston
